Card Memory Game
მოკლე აღწერა:
მეხსიერების ბარათების თამაში, სადაც მოთამაშე ბარათებს გადაბრუნებს და ეძებს ერთნაირ წყვილებს დროის ამოწურვამდე.
აღწერა:
Card Memory Game არის desktop აპლიკაცია, დაწერილი Java და JavaFX-ის გამოყენებით. მოთამაშე ირჩევს სირთულის დონეს და იღებს ბადეს დახურული ბარათებით. მიზანია ყველა წყვილის პოვნა დროის ამოწურვამდე. თამაშში არის კომბო სისტემა, ხმოვანი ეფექტები, ბარათების გადაბრუნების ანიმაცია და ერთჯერადი მინიშნების ღილაკი.
დაწყება:
წინაპირობები
Windows 10 ან უფრო ახალი
Java 17 ან უფრო ახალი
JavaFX SDK 17 ან უფრო ახალი
IntelliJ IDEA ან სხვა Java IDE
ინსტალაცია
